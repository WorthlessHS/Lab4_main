Для запуска системы управления гостиницей откройте файл решения в Visual Studio и запустите проект. Исполняемый файл приложения можно найти в папке HotelSystem\x64\Debug.

Программа позволяет полностью управлять базой гостиничных номеров через графический интерфейс. Для добавления нового номера используется специальная кнопка, которая открывает форму ввода данных. Здесь необходимо указать номер комнаты, выбрать категорию проживания между стандартным вариантом и люксом, а также задать базовую стоимость и параметры скидочной системы.

Если требуется отредактировать уже существующую запись, достаточно выбрать нужную строку в таблице и нажать кнопку изменения. При этом откроется знакомое окно редактирования, где поле с номером комнаты будет заблокировано для защиты от случайных правок, но все финансовые и категориальные параметры останутся доступными для корректировки. Удаление реализовано максимально просто: выбираете номер, нажимаете соответствующую кнопку и подтверждаете свое действие во всплывающем окне, после чего запись навсегда исчезает из списка.

Для работы с внешними данными предусмотрены функции импорта и экспорта через стандартный формат CSV. При нажатии кнопки загрузки программа предложит выбрать файл через диалоговое окно и автоматически заполнит таблицу актуальной информацией. Функция сохранения работает аналогично, позволяя выгрузить весь текущий список номеров в файл по выбранному пути.

В приложении реализована удобная система упорядочивания данных. Вы можете мгновенно отсортировать список по номеру комнаты, базовой стоимости или итоговой цене, которая рассчитывается автоматически с учетом всех скидок. Направление сортировки меняется при повторном клике на соответствующую кнопку. Кроме того, прямо под основной таблицей всегда отображается актуальная средняя стоимость проживания по всему отелю.

Программа строго следит за корректностью вводимых данных. Номер комнаты должен быть уникальным и находиться в пределах от одного до четырех девяток. Для стандартных номеров установлены ценовые границы от ста до пяти тысяч рублей с возможностью скидки до ста процентов, в то время как для люксов цена варьируется от пяти до пятнадцати тысяч, а максимальная скидка ограничена пятьюдесятью процентами. При попытке оставить поля пустыми или ввести некорректные значения система выдаст ошибку и попросит исправить данные.
