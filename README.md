Программа написана на Visual Studio, соответственно открыть можно через sln-файл. exe-файл находится по пути HotelSystem\x64\Debug.

Функционал программы:

Добавление стандартного номера. Открывается окно, в которое нужно ввести номер комнаты, базовую стоимость и выбрать тип скидки. Стандартный номер позволяет установить цену от 100 до 5000 единиц.
Добавление люкс-номера. Открывается окно, в которое нужно ввести номер комнаты, цену и параметры скидки. Для люкса предусмотрен повышенный ценовой диапазон от 5000 до 15000 единиц, а максимальный размер скидки ограничен 50 процентами.
Изменение данных номера. Нужно выбрать из списка конкретный номер и нажать на кнопку. Откроется окно, в котором можно изменить все параметры, кроме самого номера комнаты, который фиксируется для сохранения целостности данных.
Удаление номера. Нужно выбрать из списка номер комнаты и нажать на кнопку. После подтверждения выбора данные о номере удаляются безвозвратно.
Загрузить. Загружается таблица всех номеров из csv-файла. Загружаемый файл должен быть файлом, полученным при сохранении таблицы в программе, для корректного распознавания типов номеров и стратегий скидок.
Сохранить. Сохраняется вся текущая таблица номеров в csv-файл по указанному пользователем пути.
Сортировка. В программе реализованы отдельные кнопки для сортировки списка по номерам комнат, по базовой или по финальной стоимости. Также поддерживается стандартная сортировка внутри самой таблицы при нажатии на заголовки столбцов.
