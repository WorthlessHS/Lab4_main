Программа сделана в Visual Studio, открывается через sln-файл. Как пользоваться: Добавить номер. Жмем "Добавить" — выскакивает окно. Вводим номер комнаты, ставим галочку Стандартный или Люкс, пишем цену и скидку (либо без, либо процентную). Поменять номер. Выделяем строку в таблице, жмем "Изменить". То же окно, только номер комнаты серый (нельзя менять), остальное — на твое усмотрение. Удалить номер. Выбираем номер → "Удалить" → "Да, уверен" → пропал навсегда. Загрузить из файла. Кнопка "Загрузить" → ищем csv-файл (тот, что сохранили из программы). Таблица сразу обновится. Формат простой: номер;тип;цена;скидка;процент. �Сохранить в файл. "Сохранить" → выбираем куда → все номера в csv. Сортировать. Справа три кнопочки: "По ном.", "По баз.", "По итог.". Кликаем — стрелка меняется (▲ вниз/вверх), таблица перестраивается. Под таблицей всегда висит средняя цена всех номеров со скидками.
